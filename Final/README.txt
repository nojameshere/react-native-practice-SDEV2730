Here is a list of all of the assignment criteria and where they are placed so you have an easier time of looking them up.


1. Two different navigation schemes
-Tabbed navigation for the main pages
-Stack navigation for the authentication screens

2. 3 Different hooks
-useState - in virtually every single file
-useReducer - in the Context.js file
-useEffect -  in the Home and Transactions files.
-useContext - context is in most of the files

3. Database exists with functionality for user to add OR delete Database
- Database is a firestore connection storing all transactions


4. App includes user authentication
- Authenticate to access the location and photo features.


5. App uses camera and handles images
- Profile image functionality. Does not store image for database cost considerations


6. App uses map which includes markers
- Location picker for profile page.
